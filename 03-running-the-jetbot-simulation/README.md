This folder contains the custom JetBot USD file created in:

**Getting Started with Isaac Lab #3: Running a Jetbot Simulation**

## Files

- 'rc_track_params.py'
- 'jetbot_cfg.py'
- 'create_rc_track.py'
- 'run_fpv.py'

## How to use

- Copy the files into your IsaacLab\source\standalone\rc_track directory
- Edit the file paths as shown in the Youtube video
- Run each create_rc_track.py and run_fpv.py from the project root directory Isaaclab, using .\isaaclab.bat -p source\standalone\rc_track\CODE_FILE_NAME.py --experience isaaclab.python.rendering.kit
